Physics Simulation Programming: Tutorial

Main page

The following is a list of all the changable variables:

  • Camera Focus: changes the focus of the camera
  • Time: changes the number of seconds that pass every frame
  • Radius Scaling: changes the scaling of the objects radius
  • Path Length: the distance that the planets trail stays for
  • Trail radius scaling: changes the scaling of the paths radius
  • Labels: toggles the labels of all the planets
  • Gravitational constant: allows you to input a new value for the gravitational constant, controls strength of gravitational fields
  • Mass: allows you to change the mass of the object
  • Red/Green/Blue: allows you to change the RGB value of the objects
  • Radius: allows you to individually change the radius of an object
  • Trail Radius: allows you to change the radius of an objects trail
  • Orbital Incline: the angle that the objects orbit is to the xy plane
  • Point of Aphelion: the point of aphelion measured as an angle from the x axis
  • Aphelion/Perihelion distance: the distance of the objects closesed and furthest point of the object
